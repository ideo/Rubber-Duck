# Duck Duck Duck — PCB (EE)

Custom PCB designed in Eagle / Fusion 360 to hold the microcontroller and servo connector inside the duck enclosure.

## Design Files

| File | Description |
|------|-------------|
| `Ducky_Schematic.sch` | Eagle schematic |
| `DuckyBRD.brd` | Eagle board layout |
| `Ducky.f3z` | Fusion 360 project (board + enclosure integration) |
